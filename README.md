# Workflow Automation Hub
## 概要
Google Apps Scriptを使った業務自動化ツール。データ管理（スプレッドシート）、PDF生成、Slack通知を統合。データエンジニアとしてのデータフロー設計スキルをデモ。

## 機能
- **定期報告管理**: Slackからデータを取得し、Master DBに転記。
- **ドキュメント管理**: PDF生成と送付追跡（例: 在留カード）。
- **オンライン申請**: データ入力、ファイル生成、外部システム連携。
- **ダッシュボード**: リアルタイムで作業状況を可視化。

## 技術スタック
- JavaScript (Google Apps Script)
- Google Sheets/Drive API
- Slack API
- HTML/CSS (ダッシュボードUI)

## インストール
1. Google Apps Scriptプロジェクトを作成。
2. スクリプトプロパティにAPIキーなどを設定。
3. スクリプトをデプロイ（Webアプリ）。

## 設計者としての貢献
- データフロー設計（ETLプロセス）
- 自動化ロジックの提案（例: 通知タイミング、データ同期）
- ユーザーインターフェース要件定義
